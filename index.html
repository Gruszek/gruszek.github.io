Witaj w świecie HTML!

Język HTML jest podstawowym językiem używanym do tworzenia stron internetowych. Pozwala on strukturalizować treść strony za pomocą różnych tagów.

O języku HTML

HTML definiuje strukturę strony za pomocą elementów, które są określane za pomocą tagów. Każdy element może mieć atrybuty, które określają dodatkowe właściwości.

Przykładem tagu HTML jest <h1>, który definiuje nagłówek pierwszego poziomu. Inny przykład to <p>, który definiuje akapit.

Podstawowe tagi HTML

  • <h1> - Nagłówek 1
  • <p> - Akapit
  • <a> - Link
  • <img> - Obrazek
  • <ul> - Lista nieuporządkowana
  • <ol> - Lista uporządkowana
